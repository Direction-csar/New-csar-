🚀 COMMENT TESTER :
═══════════════════════════════════════════════════════════════════════

1. Ouvrir : http://localhost/csar/admin/dashboard
2. Se connecter avec identifiants admin
3. Défiler jusqu'à la section carte (en bas)
4. Observer :
   - Carte centrée sur le Sénégal
   - Marqueurs bleus (entrepôts)
   - Marqueurs rouges avec logo CSAR (demandes)
   - Statistiques au-dessus de la carte
   - Légende en haut à droite

5. Tester les filtres :
   - Cliquer sur "Filtres"
   - Sélectionner une année, un mois, une région
   - Observer la mise à jour automatique

6. Tester l'export PDF :
   - Cliquer sur "Export PDF" (bouton vert)
   - Le PDF se télécharge automatiquement
   - L'ouvrir et vérifier le contenu
