## 1. INTRODUCTION

### 1.1 Qu'est-ce que le CSAR ?

Le **CSAR** (Commissariat à la Sécurité Alimentaire et à la Résilience) est une institution publique sénégalaise dédiée à :
- La sécurité alimentaire
- L'assistance humanitaire
- La gestion des stocks stratégiques
- Le renforcement de la résilience

### 1.2 Services Disponibles

Sur ce site, vous pouvez :
- ✅ Faire une demande d'assistance
- ✅ Suivre votre demande avec votre code
- ✅ Consulter les actualités
- ✅ Télécharger les rapports publics
- ✅ Localiser les entrepôts
- ✅ Nous contacter
- ✅ S'abonner à la newsletter

## 3. FAIRE UNE DEMANDE

### 3.1 Accéder au Formulaire

**Méthode 1** : Cliquer sur "Faire une Demande" (bouton vert)  
**Méthode 2** : Menu > Demande d'Assistance  
**URL directe** : https://csar.sn/fr/demande

### 3.2 Types de Demandes

1. 🍚 **Assistance Alimentaire**
   - Denrées alimentaires
   - Aide alimentaire d'urgence
   
2. 🏕️ **Aide Matérielle**
   - Tentes, bâches
   - Matériel humanitaire

3. ℹ️ **Information**
   - Renseignements
   - Orientation

4. 📝 **Réclamation**
   - Signalement
   - Réclamation

5. 📋 **Autre**
   - Demandes spécifiques

### 3.3 Remplir le Formulaire

#### Étape 1 : Vos Informations

```
Nom complet : [Votre nom] *obligatoire*
Prénom : [Votre prénom] *obligatoire*
Email : [votre.email@example.com] *obligatoire*
Téléphone : [+221 XX XXX XX XX] *obligatoire*
CNI/Passeport : [Numéro] (optionnel)
```

⚠️ **Important** :
- Utilisez un email valide (vous recevrez la confirmation)
- Téléphone avec indicatif +221
- Toutes les informations resteront confidentielles

#### Étape 2 : Votre Localisation

```
Région : [Sélectionner] *obligatoire*
Département : [Sélectionner] (auto-chargé)
Commune : [Sélectionner] (auto-chargé)
Adresse complète : [Votre adresse]
```

**Géolocalisation** :
- 📍 Cliquer "📍 Utiliser ma position" (GPS automatique)
- 🗺️ Ou cliquer sur la carte pour placer un marqueur

#### Étape 3 : Votre Demande

```
Type de demande : [Sélectionner] *obligatoire*
Objet/Titre : [Résumé en quelques mots]
Description détaillée : [500-2000 caractères] *obligatoire*

Nombre de personnes concernées : [Nombre]
Urgence : 
  ○ Normale (7-14 jours)
  ○ Moyenne (3-7 jours)
  ○ Haute (< 3 jours)
```

**Conseils** :
- Soyez précis et détaillé
- Expliquez votre situation
- Mentionnez les besoins spécifiques

#### Étape 4 : Pièces Justificatives (Optionnel)

```
📎 Ajouter des fichiers

Types acceptés : PDF, JPG, PNG, DOCX
Taille max : 5 MB par fichier
Total max : 10 MB
```

**Documents utiles** :
- Certificat de résidence
- Photos de la situation
- Attestations diverses
- Tout document justificatif

#### Étape 5 : Validation

```
☑️ J'accepte la Politique de Confidentialité
☑️ J'accepte les Conditions d'Utilisation
☑️ Je certifie l'exactitude des informations

[Soumettre ma Demande]
```

**URL** : https://csar.sn/fr/suivi

**Option 1 - Avec code de suivi** :


**Option 2 - Sans code** :
```

```

### 4.3 Statuts de Demande

```
🟠 En Attente
   → Votre demande a été reçue
   → En attente d'attribution à un agent

🔵 En Cours de Traitement
   → Un agent a pris en charge votre demande
   → Vérification en cours

🟢 Approuvée
   → Votre demande a été approuvée
   → Préparation de l'assistance

✅ Terminée
   → L'assistance a été fournie
   → Dossier clôturé

🔴 Rejetée
   → Demande ne peut être traitée
   → Voir le motif dans les commentaires
```

### 4.4 Timeline Visuelle

```
📝 Soumise         ✅ (23/10/2025 10:30)
    ↓
📋 En Attente      ✅ (23/10/2025 10:31)
    ↓
🔍 En Traitement   🔵 (24/10/2025 14:00)
    ↓
✅ Approuvée       ⏳ (En attente)
    ↓
🎁 Terminée        ⏳ (En attente)
```

## 8. FAQ

### 8.1 Questions Fréquentes

**Q1 : Comment faire une demande d'assistance ?**  
**R :** Cliquez sur "Faire une Demande" en haut à droite, remplissez le formulaire et soumettez. Vous recevrez un code de suivi par email et SMS.

**Q2 : Combien de temps pour une réponse ?**  
**R :** Délai moyen de 7-14 jours selon l'urgence. Les demandes urgentes sont traitées en priorité (3-7 jours).

**Q3 : J'ai perdu mon code de suivi, comment faire ?**  
**R :** Sur la page de suivi, utilisez la recherche par email + nom. Ou consultez l'email de confirmation reçu.

**Q4 : Puis-je modifier ma demande après soumission ?**  
**R :** Non, une fois soumise, la demande ne peut être modifiée. Contactez-nous si vous devez apporter des précisions.

**Q5 : Ma demande a été rejetée, pourquoi ?**  
**R :** Consultez le statut de votre demande pour voir le motif. Vous pouvez soumettre une nouvelle demande ou nous contacter pour des explications.

**Q6 : Comment s'abonner à la newsletter ?**  
**R :** Formulaire en bas de chaque page. Entrez votre email et confirmez via le lien reçu.

**Q7 : Les rapports SIM sont-ils fiables ?**  
**R :** Oui, nos rapports sont basés sur des relevés terrain et validés par nos équipes.

**Q8 : Puis-je visiter un entrepôt ?**  
**R :** Les entrepôts ne sont pas ouverts au public. Contactez-nous pour des visites officielles.

**Q9 : Comment devenir partenaire du CSAR ?**  
**R :** Utilisez le formulaire de contact avec le sujet "Partenariat" ou écrivez à partenariat@csar.sn.

**Q10 : Le site est-il disponible en anglais ?**  
**R :** Oui, utilisez le sélecteur de langue en haut à droite (FR/EN).

### 8.2 Problèmes Techniques

**Le formulaire ne se soumet pas** :
1. Vérifiez que tous les champs obligatoires (*) sont remplis
2. Vérifiez votre connexion internet
3. Essayez un autre navigateur
4. Contactez-nous : support@csar.sn

**Je ne reçois pas l'email de confirmation** :
1. Vérifiez vos spams/courrier indésirable
2. Vérifiez que l'email est correct
3. Attendez 10-15 minutes
4. Contactez-nous avec votre nom et email

**La carte ne s'affiche pas** :
1. Autorisez la géolocalisation dans votre navigateur
2. Vérifiez votre connexion
3. Rafraîchissez la page (F5)
4. Essayez un autre navigateur
