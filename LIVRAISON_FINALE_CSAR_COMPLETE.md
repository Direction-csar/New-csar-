```blade
@extends('layouts.public')

    @php
        $meta = \App\Services\SeoService::generateMetaTags(
            $actualite->titre,
            $actualite->extrait,
            asset('storage/' . $actualite->image),
            'article',
            url()->current()
        );
    @endphp
    
    <title>{{ $meta['title'] }}</title>

    <meta name="twitter:description" content="{{ $meta['twitter:description'] }}">
    <meta name="twitter:image" content="{{ $meta['twitter:image'] }}">
@endsection

@section('schema_org')
    <script type="application/ld+json">
        {!! json_encode(\App\Services\SeoService::generateArticleSchema($actualite), JSON_UNESCAPED_SLASHES) !!}
    </script>
@endsection

@section('content')
    {{-- Contenu de l'actualité... --}}
@endsection
```

## 9. CONFIGURATION SERVEUR

### 9.1 Prérequis

- **PHP** : 8.1+
- **MySQL** : 8.0+
- **Composer** : 2.x
- **Node.js** : 18+ (pour Vite)
- **Extensions PHP** :
  - pdo_mysql
  - mbstring
  - openssl
  - tokenizer
  - xml
  - ctype
  - json
  - bcmath
  - fileinfo
  - gd (pour images)

### 9.2 Installation

```bash
# 1. Cloner le projet
git clone <repo-url> csar
cd csar

# 2. Installer dépendances PHP
composer install --optimize-autoloader --no-dev

# 3. Installer dépendances JS
npm install
npm run build

# 4. Configurer .env
cp .env.example .env
php artisan key:generate

# 5. Migrations et seeders
php artisan migrate --seed

# 6. Créer lien storage
php artisan storage:link

# 7. Cache config (production)
php artisan config:cache
php artisan route:cache
php artisan view:cache

# 8. Permissions
chmod -R 755 storage bootstrap/cache
```

### 9.3 Cron Jobs

**Ajouter au crontab** :

```bash
# Laravel Scheduler (toutes les minutes)
* * * * * cd /path/to/csar && php artisan schedule:run >> /dev/null 2>&1

# Backup quotidien (3h du matin)
0 3 * * * php /path/to/csar/scripts/backup/database_backup.php

# Monitoring (toutes les 5 minutes)
*/5 * * * * cd /path/to/csar && php artisan system:monitor
```
